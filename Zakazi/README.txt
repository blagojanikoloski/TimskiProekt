# Specification

## Requirements
.Net 7
Node 20.4.0
angular 16
mssql server

## Ports
api runs on port 7200
client runs on 4200


# Notes For Angular


## install npm packages
run command: npm install


## Component Creation
run command: ng g c componentname


## Route Creation
create new routes in app-routing-module.ts


## Web-Api-Client
web api calls from client are auto generated by NSwagStudio
the code is placed in services/web-api-client-service.ts
DO NOT CHANGE MANUALLY

To call a service add WebApiClient to constructor in component
and then use that instance to retrieve data from api

## Generation of Web-Api-Client with NSwagStudio
have the api runnung in the background
select runtime NetCore31
add https://localhost:7200/swagger/v1.0/swagger.json to Specification URL
click Create local Copy
select Typescript Client
Click Generate Outputs
select all, copy and paste to web-api-client-service.ts
